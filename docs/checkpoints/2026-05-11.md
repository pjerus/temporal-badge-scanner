## Workers / processes to restart on resume

None. No daemons or long-lived workers were started this session. The "worker" is the badge itself, currently running our flashed `pocket-scanner-phase1` firmware. To re-flash after code changes:

```sh
cd ~/temporal-badge-scanner/upstream/firmware
~/.platformio/penv/bin/pio run -e echo-dev -t upload --upload-port /dev/cu.usbmodem2101
# add `-t uploadfs` after `-t upload` if initial_filesystem/ changed
```

Always use `~/.platformio/penv/bin/pio` directly — system `pio` shim points to wrong Python.
